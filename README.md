# Rapport---Analyse-du-march-immobilier
L’objectif de ce projet était d’analyser les tendances du marché immobilier à travers un tableau de bord interactif réalisé sous Power BI. Le but principal était d’identifier les zones les plus rentables, de suivre l’évolution des ventes dans le temps, et de comprendre l’impact de la superficie et du type de bâtiment sur les prix de vente.
